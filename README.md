# AI投资助手 V1.0

量化辅助决策 Web 应用。支持 A 股（沪深）+ 港股通，多组合管理，IC加权量化评分，T+1/T+3 涨跌预测，历史准确率追踪（含随机基准对比）。

---

## 快速启动（Docker）

```bash
# 1. 克隆项目
git clone <your-repo>
cd ai-investor

# 2. 复制后端配置
cp backend/.env.example backend/.env
# 编辑 backend/.env，填写 SECRET_KEY（任意32+字符即可）

# 3. 一键启动
docker compose up -d

# 4. 访问
# 前端：http://localhost:80
# 后端API：http://localhost:8000/docs
```

## 本地开发启动

### 前提
- Python 3.12+
- Node.js 20+
- PostgreSQL 16

```bash
cd backend

# 安装依赖

# 复制并配置环境变量
cp .env.example .env
```

```bash

# 安装依赖
npm install

# 启动开发服务器（自动代理 /api 到 localhost:8000）
```

## 功能说明

### 仪表盘
- 实时查看自选股行情（新浪财经接口，延迟约3-5分钟）
- 量化评分（0-100分）和今日信号（偏多/中性/偏空）
- 大盘指数（上证、深证、沪深300）

### 投资组合管理
- 创建多个独立组合（如「我的A股」「妻子的A股」「港股通」）
- 搜索添加 A 股或港股通股票
- 可选填买入成本和持股数量，自动计算持仓盈亏

### 股票详情
- 近60日收盘价走势图 + 成交量
- 量化评分详情：基本面（52周位置+波动率）、技术面、资金流向、情绪面
- T+1 上涨概率 + T+3 预测区间
- 生成并记录今日预测（T+1 和 T+3 各一条）
- 历史预测记录（含实际结果对比）

### 准确率追踪
- 总体方向准确率 vs 随机基准（50%）
- T+1 / T+3 分别统计
- 月度趋势图

## 量化模型说明

### 因子体系（V1.0）

| 因子 | 类型 | IC权重 |
|---|---|---|
| 52周高低位置 | 动量（基本面代理） | +16.7% |
| KDJ_J值 | 技术面 | +17.5% |
| 5日动量 | 技术面 | +19.8% |
| 布林带%B | 技术面 | +14.3% |
| MACD柱 | 技术面 | +7.5% |
| 历史波动率 | 低波动溢价（反向） | -7.6% |
| 量比 | 资金流向 | +5.3% |
| 均线多头排列 | 技术面 | +4.9% |
| 成交量趋势 | 资金流向 | +3.3% |
| RSI(12) | 技术面 | +3.2% |

### 维度权重

- 基本面（代理）: 51%
- 技术面: 18%
- 资金流向: 15%
- 市场情绪: 15%（V1.0暂用中性值，V1.1接入新闻数据）

### 预测方法

采用**IC加权 + 截面分位数标准化**（回测验证优于旧版Sigmoid+线性归一化）：

1. 计算当日截面所有自选股的因子值
2. 分位数归一化消除极值影响（转为正态分布）
3. IC权重加权合成综合评分（0-100）
4. Sigmoid函数转换为上涨概率

回测结论（49只A股+港股通，3年历史数据）：
- 方向准确率约 **52%**，超随机基准 **+2%**
- 港股通准确率提升最显著（旧方案47% → 新方案52%）

**重要声明**：以上预测仅为量化模型参考信号，不构成投资建议。股市有风险，投资需谨慎。
