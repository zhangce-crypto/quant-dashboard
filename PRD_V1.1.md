## 1. 产品概述

### 1.1 产品定位

**AI投资助手**是一款面向家庭个人投资者的量化辅助决策 Web 应用，核心价值是将专业量化投资方法平民化——通过自动化的多因子分析模型，帮助工作繁忙、无法实时盯盘的普通投资者，在信息过载的市场环境中获得有数据支撑的投资参考信号。

本产品**不是**自动交易系统，**不是**荐股软件，而是**辅助决策工具**，所有信号仅供参考，最终决策权在用户手中。

### 1.2 核心价值主张

> 用量化的方式帮你看清每只股票的综合状态，让投资决策有据可依，而不是凭感觉。

## 2. 用户画像与痛点

### 2.1 目标用户

| 维度 | 描述 |
|------|------|
| 投资市场 | A股（沪深两市）为主，兼顾港股通 |
| 账户结构 | 共用一个系统账户，建多个独立组合分别管理 |
| 投资风格 | 中长线持股为主，偶尔短线 |
| 时间投入 | 工作日白天无法实时盯盘，利用碎片时间查看 |
| 量化能力 | 了解基本技术指标概念，无编程能力 |

### 2.2 核心痛点

- **痛点一**：无法实时跟盘，需要系统在关键时刻给出参考信号
- **痛点二**：信息过载，每天财经新闻数以百计，需要自动聚合
- **痛点三**：双账户管理割裂，缺乏统一平台
- **痛点四**：量化能力门槛高，希望有透明可理解的参考工具

## 3. 产品边界

### 3.1 不做的事

| 不做功能 | 原因 |
|----------|------|

## 4. 功能模块

### 4.1 模块总览

```
AI投资助手 V1.1
├── 模块一：用户认证
├── 模块二：仪表盘（含组合管理） ← V1.1 合并
├── 模块三：股票详情与量化分析
├── 模块四：预测准确率追踪
└── 模块五：定时任务（后台）
```

### 4.2 模块一：用户认证

- 注册（邮箱+密码）/ 登录
- JWT Token，有效期7天
- 数据完全隔离（同一账户内不同组合数据隔离）

### 4.3 模块二：仪表盘（含组合管理）

**设计决策**：V1.1 将独立的「我的组合」页面合并进仪表盘，在一个页面内完成所有组合和股票管理操作，减少页面跳转。

#### 4.3.1 组合管理（内嵌于仪表盘顶部）

| 功能 | 交互方式 | 优先级 |
|------|----------|--------|
| 切换组合 | 点击组合标签左侧名称区域 | P0 |
| 新建组合 | 点击「+ 新建组合」按钮，弹窗填名称 | P0 |
| 删除组合 | 点击组合标签右侧「×」，弹确认框 | P0 |
| 添加股票 | 点击「+ 添加股票」，弹窗搜索后选择 | P0 |
| 移除股票 | 点击股票行内「移除」按钮，弹确认框 | P0 |

#### 4.3.2 仪表盘主体

| 区域 | 内容 |
|------|------|
| 大盘指数条 | 上证指数 / 深证成指 / 沪深300，含涨跌幅 |
| 股票列表 | 评分环 / 股票名+代码 / 市场标签 / 最新价 / 涨跌幅 / 信号 / 持仓盈亏 / 操作 |

#### 4.3.3 持仓信息（可选）

添加股票时可填入买入成本和持股数量，系统自动计算持仓盈亏百分比。

### 4.4 模块三：股票详情与量化分析

访问路径：`/stock/:market/:code`，从仪表盘点击股票进入。

#### 4.4.1 左侧区域

- **近60日K线图**：收盘价折线 + 成交量柱状，ECharts 渲染
- **近期预测记录表**：点击「生成今日预测」后展示，含预测方向/概率/区间/实际结果/是否准确

#### 4.4.2 右侧区域

- **综合评分环**：0-100分，带颜色（≥65绿/40-65黄/≤40红）
- **四维评分条**：基本面/技术面/资金流向/市场情绪
- **T+1上涨概率**：与随机基准50%对比
- **T+3预测区间**：预测涨跌幅范围（%）
- **信号强度**：1-5星
- **固定风险提示**：历史回测准确率约52%，仅供参考

### 4.5 模块四：预测准确率追踪

**这是建立用户信任的关键功能。**

#### 4.5.1 全局统计（页面顶部）

| 指标 | 说明 |
|------|------|
| 总体方向准确率 | 所有已结算预测中，方向正确的比例 |
| 随机基准（固定50%） | 作为对比参照，永远显示在旁边 |
| T+1准确率 | 仅统计T+1预测 |
| T+3准确率 | 仅统计T+3预测 |

#### 4.5.2 综合准确率仪表

- 圆环图，绿色弧线表示准确率，灰色虚线弧线标注50%基准位置
- 旁边显示「超越基准 +X%」
- T+1 和 T+3 各一个小卡片单独展示

#### 4.5.3 月度趋势图

- 柱状图，按月展示准确率变化
- 绿色≥55% / 黄色50-55% / 红色<50%
- 虚线标注50%基准

#### 4.5.4 个股历史预测准确率（V1.1 新增）

- 表格：每只有历史预测记录的股票，显示 T+1/T+3准确率、预测条数、超基准幅度
- 点击股票名称展开该股历史预测明细（预测日/周期/方向/概率/实际/结果）

### 4.6 模块五：定时任务（后台）

| 任务 | 执行时间 | 说明 |
|------|----------|------|
| 刷新量化评分 | 工作日 9:30-15:00 每30分钟 | 为所有自选股重新计算评分 |
| 预测结算 | 工作日 15:35 | T+1/T+3 到期后核对实际涨跌，更新准确率 |

## 5. 量化模型（V1.1）

### 5.1 模型方案

采用**IC加权 + 截面分位数标准化**（回测对比验证优于旧版 Sigmoid + 线性归一化）。

- 方向准确率：约 **52%**，超随机基准 **+2%**
- 截面 IC 值：**0.036**（旧方案的2.9倍）
- 港股通准确率：从47%提升到52%（提升最显著）

### 5.2 因子体系

| 因子 | IC权重 | 方向 | 说明 |
|------|--------|------|------|
| 52周高低位置 | 16.7% | 正向 | 最强动量因子，IC=+0.031 |
| 5日动量 | 19.8% | 正向 | 权重最高，短期价格动能 |
| KDJ_J值 | 17.5% | 正向 | 动量指标 |
| 布林带%B | 14.3% | 正向 | 相对价格位置 |
| MACD柱 | 7.5% | 正向 | 趋势动量 |
| 历史波动率 | 7.6% | **反向** | 低波动溢价，IC=-0.022 |
| 量比 | 5.3% | 正向 | 近期成交量放大 |
| 均线多头排列 | 4.9% | 正向 | 趋势结构 |
| 成交量趋势 | 3.3% | 正向 | 最稳定，ICIR=+0.121 |
| RSI(12) | 3.2% | 正向 | 相对强弱 |

### 5.3 维度权重

| 维度 | 权重 | 说明 |
|------|------|------|
| 基本面（代理） | 51% | 52周位置+低波动，V1.1接入财报后替换 |
| 技术面 | 18% | 传统技术指标在大盘股IC较低，降权 |
| 资金流向 | 15% | 成交量趋势ICIR最稳定 |
| 市场情绪 | 16% | V1.0固定50分（无新闻数据），V1.2接入 |

### 5.4 预测转换

```
综合评分 0-100
  → Sigmoid(k=3) → 上涨概率 0-1
  → 评分≥70 = "偏多"；≤40 = "偏空"；其余 = "中性"
  → T+3预测区间 = 中值 ± 1σ（基于历史波动率）
```

## 6. 数据源

| 用途 | 接口 | 响应时间 |
|------|------|----------|
| 自选股实时价格 | 新浪财经批量接口（主）/ 腾讯财经（备） | <1秒 |
| 大盘指数实时 | 新浪财经指数接口 | <1秒 |
| 历史K线（因子计算） | AKShare 东方财富 | 3-5秒（后台预拉）|
| 港股实时 | 新浪财经 hk 前缀格式 | <1秒 |

**注意**：
- 全市场扫描接口（`ak.stock_zh_a_spot_em()`）仅用于后台定时任务，不在用户请求路径上使用（拉全量数据耗时过长）
- 行情延迟约3-5分钟，已在界面底部明确标注

## 7. 非功能需求

| 指标 | 要求 |
|------|------|
| 页面首屏加载 | ≤3秒 |
| 行情刷新延迟 | ≤15分钟（免费数据源限制） |
| 预测计算时间 | 单只 ≤5秒，全组合 ≤60秒 |

| 部署方式 | Docker Compose 一键启动（PostgreSQL+Redis+后端+前端Nginx）|
| 安全 | HTTPS，JWT认证，用户数据完全隔离 |
